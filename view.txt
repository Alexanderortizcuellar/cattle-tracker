Create a separate animal profile view that opens when selecting a cattle record instead of using the edit/create form. This profile should provide a clear, insight-driven overview of the animal, not just its basic data.

The profile should include a summary section with key information such as name, number, breed, sex, status, and age. The status should be visually highlighted and easy to understand at a glance.

For animals that are mothers, include a dedicated reproduction section. Display the total number of offspring and a table listing them with name, date of birth, status, and sale information if applicable. Each offspring should be clickable and navigate to its own profile view.

Add reproductive indicators and metrics such as time since last birth, average time between births, and age at first birth. Include logic-based indicators to classify the animal, for example identifying high-performing mothers based on number or value of offspring, detecting long periods without births, and flagging cases where the animal has not yet given birth despite reaching an expected reproductive age. Clearly show whether the animal has never given birth, has given birth, or is overdue based on defined thresholds.

Include financial insights when applicable, such as sale margin, ROI, and time to sale. These should help evaluate the animal as an investment.

Enhance the profile with contextual indicators that provide quick insights, such as reproductive performance, inactivity, or profitability signals. These should be visually distinct and easy to scan.

Add a collapsible analytics section for advanced insights, especially for mothers. This section can include charts such as offspring growth over time and value evolution of offspring, allowing deeper analysis without cluttering the main view.

The overall goal is to transform the animal view into a comprehensive profile that combines data, relationships, and intelligent indicators to support better decision-making.
