# Sistema Ganadero (Cattle Tracker)

Este proyecto es una aplicación web diseñada para llevar el control y seguimiento detallado de ganado, permitiendo gestionar animales, razas y contactos de manera eficiente.

## 🚀 Demo en Vivo
Puedes acceder a la aplicación aquí: [https://cattle-tracker-3e246.web.app/](https://cattle-tracker-3e246.web.app/)

## 🛠️ Características Principales

- **Gestión de Ganado:** Registro detallado de cada animal, incluyendo marca, número, nombre, sexo, fecha de nacimiento, raza y tipo de adquisición.
- **Control de Razas:** Catálogo personalizable de razas para categorizar el inventario.
- **Directorio de Contactos:** Registro de vendedores y compradores vinculados a las transacciones de los animales.
- **Dashboard:** Resumen visual y estadísticas rápidas del estado actual del hato.
- **Exportación/Importación:** Capacidad de respaldar y migrar toda la información en formato JSON, preservando las relaciones entre registros.
- **Sincronización en Tiempo Real:** Desarrollado con Firebase Firestore para garantizar que los datos estén siempre actualizados en todos los dispositivos.

## 💻 Tecnologías Utilizadas

- **Vue 3:** Framework progresivo de JavaScript.
- **Vite:** Herramienta de construcción rápida para frontend.
- **Vuetify 3:** Framework de componentes UI basado en Material Design.
- **Pinia:** Gestión de estado para Vue.
- **Firebase:** Autenticación y base de datos NoSQL (Firestore).
- **TypeScript:** Tipado estático para mayor robustez del código.

## 📦 Instalación y Configuración Local

1. Clona el repositorio.
2. Instala las dependencias: `npm install`.
3. Configura tus variables de entorno en un archivo `.env` (usa los datos de tu proyecto Firebase).
4. Inicia el servidor de desarrollo: `npm run dev`.
